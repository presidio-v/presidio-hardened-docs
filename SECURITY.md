# Security Policy

This repository is **documentation only** — it surfaces the public SDLC material
and an index of the Presidio-hardened projects. It ships no executable code.

## Reporting a Vulnerability

To report a security issue in any Presidio-hardened **library or service**, use
that project's own private reporting channel: open a GitHub Security Advisory in
the relevant repository (the **Security** tab → **Report a vulnerability**)
rather than a public issue.

For an issue with the documentation in this repository (e.g. a link that leaks
non-public information), open a private advisory here or contact the maintainer.

You will receive an acknowledgement within 5 business days.
